## 1. Reddit (r/thefinals)

### 英語投稿

**Title:** I built a free real-time Ranked Activity Tracker — detect cheaters before you queue into them

**Body:**

Hey everyone,

I've been working on a browser-based tool that tracks ranked players in THE FINALS by monitoring leaderboard point changes in real-time.

**What it does:**
- 🔴 **Live status detection** — See if a player is In Match, In Lobby, Offline, or Post-Match
- 📊 **Point history graphs** — Track point gains/losses over time with visual charts
- 🚨 **Cheater flagging** — Community-shared list of known cheaters, suspicious players, and notable pros
- 🌏 **Server region tags** — Mark players as AS/EU/NA so you know who's in your region
- 🔔 **Browser notifications** — Get alerted when a watched player goes into a match
- 🌐 **3 languages** — Japanese, Korean, English

**How it works:**
It polls the official leaderboard API and infers player state from point changes. If someone's points are fluctuating, they're likely in a match. No hacks, no data mining — just public leaderboard data.

**Features:**
- Add players by Embark ID (Steam/PSN/Xbox)
- Personal watchlist + community shared global list
- CSV/JSONL export for data analysis
- PWA — install it like an app on your phone
- No account required, works entirely in the browser

**Try it now (no signup):** https://thefinals-ranked-activity-tracker.yatoru19.workers.dev/

Just paste an Embark ID and hit start. Feedback welcome!
